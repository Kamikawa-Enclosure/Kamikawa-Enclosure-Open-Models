# 3D Models Library
This folder contains universal 3D solid files of IP65 aluminum extrusion enclosures.

## File Format Introduction
1. STEP: Neutral 3D format, fully compatible with SolidWorks, Fusion 360, Creo, AutoCAD and all mainstream mechanical design software.
2. STL: For rapid prototyping, 3D printing and quick shape preview.

All models match the real production dimensions of our industrial aluminum housings, support custom CNC cutting and modification.
