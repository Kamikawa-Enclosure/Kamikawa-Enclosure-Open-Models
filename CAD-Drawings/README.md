# 2D CAD Drawings Library
This folder contains 2D CAD sectional profile engineering drawings for all aluminum extrusion enclosure products.

Drawings cover the internal & external geometry of aluminum extrusion enclosures, overall dimensions, spacing of end cap screw holes, vertical clearances of PCB slots, plus full manufacturing tolerances, directly available for mechanical layout design and factory processing reference.

## Efficient Guide to Locate Target Drawings Quickly
We have hundreds of enclosure drawing files here. Downloading all files at once takes a great deal of time, and browsing files one by one also wastes your design hours. Follow this two-step lookup workflow to find your target drawing within seconds:
1. Download our specification sheet first, filter out the enclosure size, profile style and matching drawing ID you need from the sheet.
2. Enter the CAD-Drawings folder, press shortcut key `Ctrl+F` to activate the browser page search function, paste the drawing ID you retrieved, the page will jump to the exact file immediately, then you can download only the single drawing you require.
