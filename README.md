# Kamikawa-Enclosure-Open-Models
Free 3D & 2D CAD library for IP65 aluminum extrusion electronic enclosures

## 1. Product Introduction
We provide standard industrial aluminum alloy enclosures for industrial controllers, PCB boards, power supplies and embedded equipment.
All housings support CNC machining, custom cutouts, silk-screen LOGO and anodizing surface treatment.

## 2. Protection Rating
- Standard protection grade: IP65 dustproof & waterproof
- Custom IP67 upgrade available on request

## 3. File Directory & Download Guide
This repo is sorted into three independent folders for easy access:
- `/3D-Models`: STEP & STL 3D solid models, compatible with SolidWorks, Fusion 360, Creo
- `/CAD-Drawings`: DWG 2D dimension drawings for mechanical design
- `/Spec-Sheets`: PDF specification sheets with full size parameters

Click into each folder to preview or download files directly.

## 4. 3D / CAD File Usage Instruction
1. STEP files: Universal neutral format for all mainstream 3D design software
2. STL files: For rapid prototyping & 3D printing
3. DWG files: 2D engineering drawing with complete length/width/height dimension labels
