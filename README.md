# Kamikawa-Enclosure-Open-Models
## IP65 Aluminum Extrusion Electronic Enclosure Library
Industrial aluminum alloy extrusion housings with IP65 dustproof & waterproof rating, widely used for industrial controllers, PCB boards, power supplies and embedded equipment.

### Included Resources
- STEP / STL 3D printable models
- 2D CAD dimension drawings
- Aluminum enclosure selection specification sheets

### Customization Service
Custom cutout punching, surface anodizing, laser LOGO engraving, CNC precision machining, custom length & size adjustment are all supported.
